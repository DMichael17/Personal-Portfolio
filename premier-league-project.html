Premier League Match Prediction & Analytics Model

A data analytics project focused on designing a relational database and automated workflow to support predictive modeling using expected goals (xG) football performance data.

SQL Database Design SSIS Sports Analytics Predictive Modeling

Problem

Football match data is often stored across multiple sources and formats, making predictive analysis difficult without proper structure. The objective of this project was to design a normalized relational database and automation workflow that enables efficient querying, feature extraction, and preparation of datasets for predictive modeling.

Tools & Technologies

SQL Server SSIS Python Understat API Football Data Data Modeling

My Role & Contributions

  • Designed normalized database schema for teams, matches, and expected goals data.
  • Imported and validated historical Premier League match datasets.
  • Created SQL queries to support predictive feature generation.
  • Developed SSIS automation package to execute model workflow.
  • Integrated relational database outputs into predictive modeling scripts.
  • Tested model performance using completed matchday data.

Implementation

Data was structured into relational tables linked through primary and foreign keys. An automated SSIS package was built to refresh prediction datasets and execute model processes with minimal manual intervention. This workflow supports scalable analytics and repeatable prediction cycles.

Results

  • Built automated analytics pipeline for football match prediction.
  • Organized raw sports datasets into structured relational model.
  • Demonstrated ability to combine database engineering and predictive analytics.
  • Established foundation for future machine learning performance improvements.
